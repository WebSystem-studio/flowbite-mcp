🔧 How to Test with Cline Extension

  1. Open VS Code Command Palette (Ctrl+Shift+P)
  2. Run: Cline: Open MCP Settings
  3. Add our MCP server configuration (see CLINE_SETUP.md)
  4. Open Cline extension panel
  5. Try prompts like:
    • "Generate a primary button with text 'Click Me'"
    • "Create a contact form component"
    • "Validate this HTML: <button class='bg-blue-500'>Test</button>"
    • "Suggest components for an e-shop product page"
