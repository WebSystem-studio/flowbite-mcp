🔧 How to Test with Cline Extension

  1. Open VS Code Command Palette (Ctrl+Shift+P)
  2. Run: Cline: Open MCP Settings
  3. Add our MCP server configuration (see CLINE_SETUP.md)
  4. Open Cline extension panel
  5. Try prompts like:
    • Buttons: "Generate a primary button with text 'Buy Now'"
    • Inputs: "Generate an email input with label 'Email Address' and required validation"
    • Search: "Create a search input for product search"
    • Textarea: "Generate a resizable textarea for user messages"
    • Alerts: "Create a success alert saying 'Data saved successfully'"
    • Forms: "Create a contact form component"
    • Suggestions: "Suggest components for a user registration form"
    • Validation: "Validate this HTML: <input type='email' class='border'>"
